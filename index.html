Brief description

A user-friendly and object-oriented API for reading and writing JSON files, written in modern Fortran (Fortran 2003+). The source code is a single Fortran module file (json_module.F90).

License

The JSON-Fortran source code and related files and documentation are distributed under a permissive free software license (BSD-style). See the LICENSE file for more details.
