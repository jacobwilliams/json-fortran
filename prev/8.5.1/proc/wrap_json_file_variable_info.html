private subroutine wrap_json_file_variable_info(me, path, found, var_type, n_children, name)

Alternate version of json_file_variable_info, where “path” is kind=CDK.

Note

If found is present, no exceptions will be thrown if an error occurs. Otherwise, an exception will be thrown if the variable is not found.

    subroutine wrap_json_file_variable_info(me,path,found,var_type,n_children,name)
